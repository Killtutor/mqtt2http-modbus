Utilice las opciones de seguridad para proteger el acceso al sistema contra usuarios no autorizados.

Si un usuario sobrepasa el límite de contraseñas erróneas será bloqueado por el sistema. Para desactivar el bloqueo de usuarios debe seleccionar un límite igual a cero.

Desbloquear usuarios luego de indica el tiempo que debe esperar un usuario para poder volver a intentar ingresar al sistema luego de sobrepasar el límite de contraseñas erróneas. El desbloqueo manual indica que el acceso no será restituido automáticamente, por lo que se debe contactar a un administrador para restituirlo. El acceso de los administradores siempre será restituido automáticamente.

La autenticación de captcha permite diferenciar un ser humano de una máquina, ayudando a evitar ataques de sistemas automatizados.
