Overview

This data source is used to collect data sent from persistent TCP publishers on other Mango instances. (The data source is capable of receiving publisher connections from the same instance, but this is typically of little utility.)

Configuration

Every data source requires a Name, which can be any description. The XID is used in the import/export facility, and must be unique among all data sources. By default a value is automatically generated that is unique at the time of generation.

The Port is the TCP port number at which the data source will listen for connection requests from publishers. This can be any number from 1 to 65535, but some care must be taken to choose a number that is not already in use on the host.

The Authorization key is a shared secret key between the publisher and the data source, which provides security on the connection by preventing unauthorized connections to the data source.

The Accept point attribute udpates checkbox determines whether point update requests from the publisher should be accepted and used. For example, if a user on the publisher system changes the name or other properties of a point configured in the publisher, the publisher will notify the data source of the change. If such changes are accepted the change will be implemented on the data source instance as well.
