Se pueden crear listas de correos para manejar fácilmente múltiples direcciones de email a la vez. Se puede hacer referencia a los correos de usuarios o agregarlos de forma explícita. Estas listas se utilizan para contener los destinatarios de correo electrónico dentro del sistema.

Para crear nuevas listas de correos debe hacer clic en el icono . Para editar una lista existente debe seleccionarla haciendo clic en su nombre.

Para agregar un usuario o dirección de correo de forma explícita debe hacer clic en el icono correspondiente a cada casilla.(Nota: cuando un usuario modifica su dirección de correo en la sección de Usuarios estos cambios también se harán efectivos en la lista de correos si ha sido agregado como entrada usuario y no como entrada explícita). Una entrada de usuario se representa con el icono de usuario (dependiendo del tipo de usuario), una entrada explícita se representa por el ícono . Para remover una entrada de la lista se debe hacer clic en el icono correspondiente a la misma.
