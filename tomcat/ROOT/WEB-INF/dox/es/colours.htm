Este sistema provee múltiples lugares donde los colores pueden ser definidos. En general, se aplican estas reglas donde los colores son definidos como cadenas de texto. Se puede introducir cualquier valor de color CSS. Solo el valor del color debe ser proporcionado. Por ejemplo, los valores válidos para el color rojo serían (excluyendo las comillas) "#FF0000", "red", "#F00", "rgb(255, 0, 0), y "rgb(100%, 0%, 0%)". Puede encontrar una descripción simple de valores de colores aquí. Para información más específica visitar W3C
