Vision general

En cada actualización de punto (o cambio, según corresponda) el publicador envía a los servidores de Cosm los valores API key, feed, ID y data stream ID como peticiones HTTP separadas.

Propiedades

La API Key es requerida para actualizar datos privados. Puede encontrarla en el área de configuración dentro de su cuenta de Cosm. Los parámetros de Timeout y Reintentos son utilizados para controlar errores.

Respuesta HTTP

Cualquier respuesta difetente de "200 OK" es considerada un error y se activará una alarma. Note que dependiendo de la cuenta de Cosm, se pueden aplicar algunas restricciones en la respuesta. Ver https://cosm.com/docs/ para mayor información.

Puntos del publicador

La sección de Puntos muestra los puntos a ser publicados. La publicación de puntos puede ser habilitada () o deshabilitada (). Los valores Feed ID y Data stream ID determinan los datos que serán actualizados.

Para agregar un punto a la lista, seleccionarlo y hacer clic en el icono . Para borrar un punto existente, hacer clic en el icono correspondiente al punto.
