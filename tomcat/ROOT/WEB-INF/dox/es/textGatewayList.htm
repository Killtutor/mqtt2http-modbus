La lista de dispositivos de mensajería muestra todos los dispositivos definidos en el sistema para enviar mensajes de texto; mostrando el nombre, tipo, configuración, estado y reglas de envío de cada uno. La configuración depende del tipo de dispositivo. El estado determina si el dispositivo está habilitado () o deshabilitado (). Haciendo clic en estos iconos se cambia el estado del dispositivo.

Para agregar un nuevo dispositivo de mensajería seleccionar su tipo y luego hacer clic en el icono en la parte superior de la lista. Para modificar un dispositivo existente, hacer clic en el icono asociado al mismo. Para borrar un dispositivo, hacer clic en el icono asociado al mismo.

Las flechas ( ) determinan el orden de prioridad de cada dispositivo. La lista se ordena de mayor a menor orden de prioridad.
