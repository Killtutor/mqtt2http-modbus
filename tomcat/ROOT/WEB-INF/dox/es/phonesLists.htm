Se pueden crear listas de teléfonos móviles para manejar fácilmente múltiples números telefónicos a la vez. Se puede hacer referencia a los números telefónicos de usuarios o agregarlos de forma explícita. Estas listas se utilizan para contener los destinatarios de mensajes de texto (SMS) producidos por las alarmas del sistema.

Para crear nuevas listas de teléfonos móviles debe hacer clic en el icono . Para editar una lista existente debe seleccionarla haciendo clic en su nombre.

Para agregar un usuario o número telefónico de forma explícita debe hacer clic en el icono correspondiente a cada casilla.(Nota: cuando un usuario modifica su número telefónico en la sección de Usuarios estos cambios también se harán efectivos en la lista de teléfonos móviles si ha sido agregado como entrada usuario y no como entrada explícita). Una entrada de usuario se representa con el icono de usuario (dependiendo del tipo de usuario), una entrada explícita se representa por el ícono . Para remover una entrada de la lista se debe hacer clic en el icono correspondiente a la misma.
