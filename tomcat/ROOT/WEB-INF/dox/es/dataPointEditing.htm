Esta página proporciona controles para la edición de las propiedades de un punto. El icono provee un enlace a la página de detalles del punto; sólo será visible si el punto se ha guardado previamente.

El Nombre del punto es requerido y puede contener cualquier texto. Este nombre será usado como identificador visual del punto dentro del sistema.

Frecuentemente el nombre de punto se coloca junto con el nombre del dispositivo cuando este se muestra en las listas con otros puntos. Esto puede proporcionar un identificador único dentro de la instancia del sistema, algo especialmente útil cuando se tienen múltiples puntos con el mismo nombre. El "nombre del dispositivo" es fijado inicialmente con el nombre de la fuente de datos, pero puede ser cambiado.

El nombre de la fuente de datos se muestra como referencia. El icono es un enlace al panel de configuración de dicha fuente de datos, donde se incluyen las propiedades de bajo nivel de cada punto.
