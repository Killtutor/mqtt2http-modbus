Visión general

Los mantenimientos programados son usados para programar periodos de mantenimiento para fuentes de datos, de tal manera que los eventos generados por la fuente de datos y sus puntos dentro de este período son automáticamente reconocidos y no se envían alarmas por correo electrónico o SMS. Cuando se inicia el modo de mantenimiento, se activa un evento único, este evento se desactiva cuando vuelve a la normalidad o es desactivado manualmente. Observe que se puede activar manualmente el modo de mantenimiento de ser necesario.

Listas de mantenimientos programados

Para agregar un nuevo mantenimiento programado, hacer clic en el icono . Para modificar un evento de mantenimiento existente, hacer clic en .
La primera columna de la lista muestra el icono cuando el evento programado se encuentra habilitado, en caso contrario se muestra ; haciendo clic en estos iconos se puede habilitar/deshabilitar directamente sin tener que acceder a la configuración.
En la columna estado se muestra el icono cuando se encuentra activo el modo de mantenimiento y para inactivo. Cuando la planificación se realiza de forma manual el modo de mantenimiento estará activo siempre que esté habilitado el mantenimiento programado.

Edición de eventos de mantenimiento

La Fuente de datos es aquella a la que se desea realizar mantenimiento.

El Alias es un valor opcional que será usado en la descripción mantenimiento programado dentro del sistema si este es proporcionado. En caso contrario, se usará una descripción generada automáticamente.

El Nivel de alarma determina si se genera una alarma al activarse el modo de mantenimiento y la gravedad de la misma. Consulte la página de ayuda general para obtener información sobre las alarmas.

El Tipo de planificación determina el periodo de tiempo en que se realizará el mantenimiento. Esos tipos incluyen horario, diario, etc. Por ejemplo, un tipo de planificación horario correrá el modo de mantenimiento cada hora. Si se requiere realizar mantenimiento dos veces al día, es necesario definir dos mantenimientos programados diarios, uno para cada hora. Sin embargo, también es posible definir un patrón cronológico que corra varias veces dentro del periodo de tiempo. (Ver la documentación "Patrones de cronológicos" para más información)

Para habilitar o deshabilitar un mantenimiento programado hacer clic en el icono o según sea el caso. Cuando la planificación se realiza de forma manual el modo de mantenimiento estará activo siempre que esté habilitado el mantenimiento programado

Horas de Actividad/Retorno

Los campos disponibles para la edición de horas de actividad y retorno dependen del tipo de planificación seleccionado. Estos campos no deberian requerir explicación ya que son basados exclusivamente en el calendario gregoriano.

Para guardar un mantenimiento programado, hacer clic en el icono en la parte superior derecha. Para borrar un mantenimiento programado existente, hacer clic en el icono .
