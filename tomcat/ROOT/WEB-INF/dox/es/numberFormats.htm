Los formatos numéricos son utilizados para determinar la sintaxis en que se muestran los valores numéricos, convirtiendo valores de punto flotante o enteros a su representación en modo de texto. Por ejemplo, el número 1234.5678 puede ser mostrado como se ve, pero sería mejor visualizado como '1.234,6', o '1234,567', o '1,2345678E3' dependiendo del contexto y semántica del valor. Los mismos patrones pueden ser usados para dar formato a textos para convertirlos en valores numéricos.

El formato numérico en este sistema es implementado por la case Java DecimalFormat, para mas detalles hacer click aquí. Algunos ejemplos pueden ser:

  • #.##0,00 Se usa el punto como separador de miles, coma para los decimales. siempre se muestra el primer dígito entero y dos valores decimales (redondeados).
  • 0,000 Siempre se muestra el primer dígito entero y tres decimales.
  • 0,00;(0,00) Siempre se muestra el primer dígito entero y dos decimales. Los valores negativos se muestran entre paréntesis.

El formato puede ser determinado dependiendo de los valores por defecto de la Máquina Virtual de Java.
