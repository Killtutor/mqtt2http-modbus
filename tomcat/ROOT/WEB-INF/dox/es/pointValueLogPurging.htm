La funcionalidad Borrar ahora permite un borrado fácil de históricos en base a su edad. Los valores que se borran se eliminan permanentemente de la base de datos y son irrecuperables. Use los controles de Borrar datos de hace mas de o Borrar todos los datos y presione Borrar ahora. El borrado se produce de forma asincrónica, por lo que puede seguir navegando por el sistema, pero en este caso, no recibirá la notificación cuando el proceso haya concluido.
