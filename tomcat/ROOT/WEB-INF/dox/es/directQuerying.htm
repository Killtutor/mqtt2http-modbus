El sistema provee la habilidad de acceder directamente a su base de datos tanto por consultas como actualizaciones. Esta funcion se provee principalmente para la resolución de problemas o reparación del sistema. No es apropiada para uso regular por producir un gran riezgo de corrupción de datos.

Sólo se pueden enviar sentencias SQL válidas. No terminar las sentencias con el caracter ';'.
