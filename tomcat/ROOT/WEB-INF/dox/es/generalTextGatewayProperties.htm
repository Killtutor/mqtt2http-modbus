Visión general

Los dispositivos de mensajería son el medio por el cual se envían los mensajes de texto generados por el sistema.

Propiedades generales

Cada dispositivo de mensajería tiene un Nombre que es usado para identificarlo visualmente en la lista. Se recomienda que cada dispositivo tenga un nombre único, sin embargo no es requerido. La casilla Habilitado es utilizada para habilitar o deshabilitar el dispositivo.

Se puede incluir una lista de códigos telefónicos para los cuales el dispositivo tiene prioridad. Para agregar un código debe incluir un nombre para identificarlo (por ejemplo el nombre de la operadora de telefonía), el código asociado (por ejemplo el código de dicha operadora) que debe estar en formato de números internacionales (empezando por el símbolo '+' junto con el código del país asignado por la ITU) y hacer clic en el icono dentro de la lista de Códigos de teléfonos prioritarios

Para bloquear un número o código determinado debe agregarlo a la lista Códigos de teléfonos no permitidos en la misma forma que en la lista de códigos de teléfonos prioritarios.
