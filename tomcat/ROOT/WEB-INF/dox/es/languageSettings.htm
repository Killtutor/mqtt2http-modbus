Utilice la configuración de idioma para determinar el idioma que el sistema utilizará cuando este sea desconocido. Por ejemplo, se utiliza este idioma cuando se generan informes y correos que están asociados a múltiples usuarios con idiomas diferentes.

Tenga en cuenta que el sistema utiliza algunas herramientas que no son de i18n, por lo que aún puede recibir mensajes en inglés.
