Phones lists can be added to easily manage multiple mobile phones at once. They can reference the mobile phone of users as well as contain explicit (external) phones. These lists can then be used a recipients for messages sent by SMS event handlers.

Use the icon to create a new phones list, or edit an existing list by clicking it's name in the list.

When editing a list, use the icon to append the user's or explicit phone to the list. (Note that changes to the user's mobile phone when editing a user will also update the phones list.) A user entry in the list is designated with a user icon (depending on the user's type); an explicit phone is listed with a . To remove an entry in the list, click the icon associated with the entry.

To save your changes to a list, click the icon. To delete an entire list, click the icon. To send a test SMS to all of the current entries in the list, click the icon.
