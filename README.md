# mqtt2http-modbus

mqtt2http-modbus is a software gateway to translate the MQTT protocol to HTTP and modbus over TCP

## Installation

clone the repo
Install Node
npm install


```bash
apt-get install node 
npm install
```

## Usage

```bash
node init.js
```
